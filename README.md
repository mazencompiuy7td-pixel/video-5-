# Mazen — Video Downloader

مشروع بسيط يُمكّن المستخدم من إدخال رابط من TikTok / Facebook / Instagram ثم تنزيل الفيديو عبر خادم يستخدم `yt-dlp`.

**محتويات**
- `server.js` — خادم Node.js (Express) للتعامل مع yt-dlp.
- `index.html` — واجهة المستخدم (محدثة لتجميل الواجهة).
- `style.css` — تصميم عصري RTL.
- `app.js` — منطق الواجهة: جلب الرابط المباشر وتنزيل مع شريط تقدم.

**تشغيل محلي سريع**
1. ثبت متطلبات الخادم: `node` و `yt-dlp` و `ffmpeg`.
2. في مجلد المشروع شغّل:
   ```
   npm install
   node server.js
   ```
3. افتح `index.html` أو زر `http://localhost:3000` لو قمت بخدمة الملفات الثابتة.

**ملاحظة قانونية**
استخدم الأداة بمسؤولية. تنزيل أو إعادة نشر محتوى محمي بدون إذن قد يخالف القوانين وشروط المنصات.
